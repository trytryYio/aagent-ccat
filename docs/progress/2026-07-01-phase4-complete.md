## 🔧 技术实现细节

### 槽位填充流程
```
用户输入 → 规则提取（快速）
         ↓ 缺失关键字段
         LLM 兜底（复杂表达）
         ↓
slots = {budget_max, category, scenario, gender, foot_type, tech_preferences}
```

### 结构化过滤流程
```
slots.category → Qdrant category_filter（检索时过滤）
slots.budget → 检索后过滤（底层不支持 price filter）
         ↓ 过滤后结果 < 2
         保留原始结果（避免完全没结果）
```

### 多因子澄清门控
```python
if top_score < 0.6:
    强澄清（低置信）
elif top_score < 0.75 and gap < 0.05 and not user_provided_clear_slots:
    弱澄清（难分伯仲 + 槽位不完整）
else:
    不澄清
```

### 答案生成增强
```python
system_prompt = f"""
输出要求：
1. 显式引用：推荐商品时必须提到商品名称
2. 引用来源：说明来自哪个候选商品或知识引用
3. 推荐解释：结合用户槽位解释推荐理由
4. 不确定表达：信息不足时说"这部分信息我未能确认"
5. 不要编造：不要推荐候选列表之外的商品
"""
```

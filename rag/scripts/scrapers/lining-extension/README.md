# 李宁官网 Chrome 扩展 - 商品数据采集

**永不触发反爬** - 使用浏览器扩展方式，在用户正常浏览时自动捕获商品数据。

## 原理

用户正常浏览李宁官网，扩展在后台无声工作：
- 拦截列表 API - 捕获商品基本信息
- 拦截详情 API - 捕获详细信息
- 不需要模拟登录（使用用户自身会话）
- 不需要代理 IP（复用用户自身 IP）
- 不触发任何反爬（请求是用户自己发的）

## 安装

1. 打开 Chrome 浏览器
2. 访问 chrome://extensions
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 lining-extension 文件夹

## 使用方法

### 1. 采集列表数据

打开李宁官网分类页面：
- 男鞋: https://store.lining.com/goods/list?category=运动鞋_男鞋
- 女鞋: https://store.lining.com/goods/list?category=运动鞋_女鞋

浏览商品列表（滚动页面触发加载更多），扩展自动捕获商品信息。

### 2. 采集详情数据

点击商品进入详情页，扩展自动捕获详细信息：
- basic_info（基础信息）
- introduction（产品介绍）
- detail_images（详情图片）

### 3. 导出数据

1. 点击浏览器右上角扩展图标
2. 查看已捕获数据统计
3. 点击「导出 CSV」或「导出 JSON」
4. 选择保存位置

## 优势对比

| 方案 | 反爬风险 | 维护成本 | 数据质量 |
|------|---------|---------|---------|
| Playwright 模拟浏览器 | 易被检测 | 高 | 中 |
| Requests 直接请求 | 被403拦截 | 高 | 低 |
| **Chrome 扩展** | **永不触发** | **低** | **最高** |
